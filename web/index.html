ABHAY'S CHIP-8 EMULATOR

SYSTEM_ARCHITECTURE

CHIP-8 is a virtual machine originally designed in the mid-1970s. This project is a hardware-accurate emulator that replicates the computer architecture of early microcomputers. It functions by manually managing 16 general-purpose 8-bit registers, a 12-bit Program Counter (PC), and a stack for subroutines—effectively simulating the fetch-decode-execute cycle of a physical CPU.

The emulator core is written in C++ and uses SDL3 for low-level graphics, audio, and input handling. To bring this system to the web, the codebase is compiled into WebAssembly (Wasm), allowing the emulated logic to run with near-native performance inside the browser.
